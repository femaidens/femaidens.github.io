Reference in Documentation

  • To log data, we use the DataLogManager class
  • Usually use this in the Robot.java class in the robotInit() method using the DataLogManager.start(); method
  • This creates logs, initialized at the start of every ENABLE
  • Logs will be stored in the RoboRIO memory
  • Can be accessed through the following steps: (refer to SysID, covers them)
  • VSCode > WPILib: Start Tool > DataLogTool
  • Download > Team Number: 2265; Username: lvuser > Connect
  • Input Files > will display what files are currently stored
  • Files from the bottom of the file list is the most recent
  • Select them > Download, delete after downloading if you want
