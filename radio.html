How can we find the IP address for direct and quicker connection?

  1. Go to Terminal/Command Line/Powershell
  2. Type: ping [insert-url-address]

Upgrading firmware:

NOTE: MAKE SURE ALL AUX SWITCHES ARE TURNED OFF OR YOU MIGHT CORRUPT THE RADIO

  1. Download the most updated file on the firmware releases page
  2. Enter the corresponding SHA-256 listed on this web page for the corresponding firmware file that you downloaded
  3. Click "Upload"
  4. Wait until system status (SYS) light blinks slowly

Note for resolving connection issues: The radio should not be fully enclosed by a case or mounting solution. This traps heat in the radio and can cause the radio to overheat. When the radio is overheating the latency (time for data to travel from your driver station to the robot) will increase.

Tethering for competitions: (on the practice field)

Access Point AND Robot Radio

This is when you have two radios that are ‘talking’ to each other. This allows you to connect to the robot wirelessly with a stronger connection than the 2.4GHz, a.k.a. the 6GHz frequency.

IN ORDER TO DO THIS, MAKE SURE THAT YOUR 2.4GHz switch on both radios are turned off!

Configure one radio as the Robot Radio once you reach the Radio configuration page

  1. Input team number [2265]
  2. SSID Suffix is the name that you want to name your radio
    1. It has to be the same for both radios!!
    2. Currently (2024-2025), the name is gloE
  3. WPA key for 6GHz connection
    1. Just put ‘vividhosting’
    2. Its just the password
  4. WPA key for 2.4GHz connection
    1. Same as above
    2. Just the password
  5. Click configure

Configure the other one as the Access Point Radio; this will be the one you only plug into a power source: Label the radios, you cannot mix them up.

  1. Input team number [2265]
  2. SSID Suffix is the name that you want to name your radio
    1. It has to be the same for both radios!!
    2. Currently (2024-2025), the name is gloE
  3. WPA key for 6GHz connection
    1. Just put ‘vividhosting’
    2. It's just the password
  4. It’ll ask for a channel. You want to use one that is known to work. There are a lot that just don’t work, but I think channel 21 works.
  5. Click configure

The robot radio is the one that is connected to the electronics board/is physically on the robot.

The access point is the one that you only supply power to.

You can connect to the radio connection wirelessly through the network!

You know that they are connected/’talking to each other’ when both of them have the 6GHz lights on at the same time.

If not, it might be broken. Please check that. And confirm that you know which one is actually tweaking out.

NOTE: It’s illegal to wire the robot radio without a POE!! Use. A. POE.. ( You might break the radio )
