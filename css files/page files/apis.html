APIs

Application Programming Interface or APIs serve as documentation for the libraries we use. When in doubt of what methods to use or what parameters to put, check the API.

Below are the links to the 2025 APIs. They typically update every year, so you will need to look for the new ones every year!

PathPlanner API:
https://pathplanner.dev/api/java/

New Rev API:
https://codedocs.revrobotics.com/java/com/revrobotics/package-summary.html

New WPILib API:
https://github.wpilib.org/allwpilib/docs/release/java/index.html

Limelight API:
https://docs.limelightvision.io/docs/docs-limelight/apis/complete-networktables-api

PhotonVision API:
https://javadocs.photonvision.org/
