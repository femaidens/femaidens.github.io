Driver Station

The Driver Station is the program we use to communicate our code to the robot.

Driver Station Essential Tips(unfinished)

  • Correctly indicate whether teleop or autonomous testing is applicable
  • Joysticks Red Bar = Confirm the joystick ports and the connection of the joysticks to Driver Station
  • Robot Code Red Bar = Confirm deployment and build of code (Connection to WiFi is necessary for building and connection to radio is necessary for deployment). Reboot radio code is available on the second tab of Driver Station
  • Communication Red Bar = Identify connection with radio and reboot Roborio is available on the second tab of Driver Station
  • Battery Voltage: The battery voltage will be red when the Roborio brownout is triggered (Battery should be changed between rounds
  • Gear Icon in Drivers Station

Testing

Before you press teleoperated, there are things you NEED to watch out for:

  1. Are your driver station joysticks plugged into your computer?
  2. Did you pull from your repository and deploy it?
  3. Are your ports correct for all your motors?
  4. Make sure the speed of the mechanisms are not super fast
  5. Can you connect to the radio?
  6. MAKE SURE YOU OR SOMEONE ELSE CAN PRESS ENTER IN CASE SOMETHING GOES WRONG TO DISABLE THE ROBOT
