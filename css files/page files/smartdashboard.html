SmartDashBoard

SmartDashboard displays robot data in real-time while the program is running in the form of graphs, text, dials, etc.

To display data, you call specific SmartDashboard methods in the periodic section of the subsystem that reference “getter” methods, which return numeric values, or boolean methods that return the status of a mechanism. Boolean and numeric values each have a specific method to write their data to the SmartDashboard, which is simply .putNumber() or .putBoolean. Each method takes a String message and a method that returns the value it will display as a parameter.
