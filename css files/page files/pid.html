PID

What is PID? PID is a feedback controller that consists of proportional, integral, and derivative terms. It is a closed loop system that allows the robot to get to a setpoint as accurately as possible with minimal time.(You can look into the math as it helps you understand how it works and adjust PID values but knowing this definition is definitely enough)

For example you need PID for an elevator to extend up to close to 4 feet.
