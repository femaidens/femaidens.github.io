REV Hardware Client

The REV Hardware Client detects connected REV devices and makes it easier for the user to update them.

We usually use the client to change the settings for SparkMaxes connected to motors. To use the REV Hardware Client, open the application and connect the REV Device through the orange wire(USB).

Make sure that the client is updated to the most recent version by downloading updates.

Once the REV Device is connected to the computer, scan for devices and the user should be able to see the device ID on the hardware page along with other information about the device by clicking onto it. The user can directly change the settings of the device through the client.

CTRE Client

Phoenix Tuner X is the companion application allowing you to update, configure, analyze, and control your CANcoders, Pigeons and Talons (Krakens).

Installation: Microsoft Store, Google Play Store, Apple App Store

Connecting to the Robot: In the bottom right corner, click the downwards arrow to release a drop-down box. Click Driver Station for the correct version and open the actual Driver Station software.

Device List: The Devices page is the first page that is shown to the user upon launching the application.

The color of the device cards is helpful as a visual indicator of device state. The meaning of the card color is also shown as text underneath the device title.

Device History: Allows teams to view previously connected devices and license them without being directly connected to them.

Device Details: When clicking into each device, you load into the Device Page. This page is where you can edit the IDs, name each device, update each device and blink the devices (Identifies each device by creating a rapid blinking effect). To save all your changes, click set and blink if you are unsure of the device changes.

Running Devices: When running Talons (Including the swerve modules), ensure that the control mode is on DutyCycleOut. Click the RED DISABLED button to enable Phoenix Tuner and ENABLE on DriverStation. Then, you can use the slider output to manipulate the voltage inputted to the Talons. Disable in both softwares when finished.

Plotting: Play the Plot page for real-time logging and click the Reset Zoom button to clearly see where your logging is starting.
